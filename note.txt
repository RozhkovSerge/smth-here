1. git restore . //возвращает все непроиндексированные файлы в исходное состояние. Аналог git reset --hard
2. git restore --staged // переводит файлы из staged в unstaged состояние (из зеленого в красное)
3. git clean -fd // удаляет файлы и директории из рабочей директории, которые не были проиндексированы. 
4. git revert commithash // создание коммита отменяющего комит с указанным хэшем
5. git reset --hard HEAD~ // удаляет последний коммит
6. git reset --hard HEAD~2 // удаляет 2 последних коммита
7. Удалять комиты следует осторожно, желатьельно чтобы они не были запушены на сервер.
